JVM – Java Virtual Machine – основная часть JRE, то, что непосредственно выполняет код Java; содержит в себе JIT-калькулятор (то есть систему, которая находит куски кода, которые нужно подвергнуть JIT-компиляции), сборщик мусора (Garbage Collector), система управления и создания потоков и прочие вспомогательные инструменты. Виртуальная машина Java исполняет байт-код Java, предварительно созданный из исходного текста Java-программы компилятором Java (javac). Использование одного байт-кода для многих платформ позволяет описать Java как «скомпилировано однажды, запускается везде» (compile once, run anywhere).
