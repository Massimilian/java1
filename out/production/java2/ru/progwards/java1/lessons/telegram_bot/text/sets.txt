Set (множества).
	К множествам относятся наследные от Set классы (который сам наследуется от Collection). Множество – это коллекция, состоящая из уникальных элементов. Основные методы интерфейса Set:
•	int size() – количество элементов внутри множества;
•	boolean isEmpty() – проверка на пустоту;
•	boolean contains(Object o) – проверка на наличие конкретного объекта внутри множества;
•	Boolean containsAll(Collection<?> c) – проверка на наличие ВСЕХ объектов внутри множества;
•	Object[] toArray(), <T> T[] toArray(T[] a) – сформировать массив из элементов множества;
•	boolean add(E e) – добавить элемент во множество;
•	boolean addAll (Collection<? extends E> c) – добавить все элементы из коллекции c;
•	boolean remove(Object o) – удалить объект (внимание! сравнение через equals());
•	boolean removeAll(Collection<?> c) – удалить коллекцию объектов (внимание! сравнение через equals());
•	boolean retainAll(Collection<?> c) – удалить объекты, кроме указанных;
•	void clear() – очистить коллекцию.
	Особые статические методы множества (Set):
•	Set<E> Set.of (E …) – формирует множество из объектов e1, e2, … . При этом само множество будет Unmodifiable (то есть само множество неизменяемо – туда нельзя добавлять и оттуда нельзя удалять элементы, но сами элементы могут меняться).
•	Set<E> Set.copyOf(Collection<? extends E> c) – формирует множество из элементов коллекции. При этом само множество будет Unmodifiable (то есть само множество неизменяемо – туда нельзя добавлять и оттуда нельзя удалять элементы, но сами элементы могут меняться). Изменения внутри самого оригинала не повлияют на копию.

Пример использования:

public class Test {
    public Set<String> worker(String... strings) {
        System.out.println("Number of parameters: " + strings.length); // выводит количество параметров
        return Set.of(strings); // формируем Set
    }

    public static void main(String[] args) {
        Test test = new Test();
        System.out.println(test.worker("One", "Two", "Three", "Four")); // Превращение строк в Set. Внимание - должны быть оригинальные значения!
    }
}
