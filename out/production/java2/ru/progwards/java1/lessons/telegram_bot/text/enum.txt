Синтаксис enum существует для перечисления различных ограниченных по количеству сущностей. Пример объявления enum:
enum DaysOfWeek {
    SUNDAY, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAT, SATURDAY
}

Элементы enum принято именовать большими буквами, так как, по сути, они являются константами. Присвоение в переменную значения enum осуществляется так:
DaysOfWeek dayOne = DaysOfWeek.SUNDAY; 

Другими словами, enum – это перечисление возможностей, которые может принять этот самый enum. Далее в переменную мы можем присвоить одно из значений этой самой переменной. Для enum обычно создаётся отдельный класс.
