Process - это абстрактный класс, определяющий, какие методы должны быть в реализациях для конкретных платформ. Методы:
getInputStream() / getErrorStream() / getOutputStream() – методы, которые возвращают стандартные потоки сообщений о вводе, ошибках и выводе.
void destroy() -  уничтожение процесса.
int exitValue() – возврат кода завершения процесса (по умолчанию 0 – это нормальное завершение).
int waitFor() - вынуждает текущий поток выполнения приостановиться, пока не будет завершен процесс, представленный этим экземпляром Process; возвращает значение кода завершения процесса.
