День рождения Java – 23/05/1995. Главный родитель – Джеймс Гослинг (Sun Microsystem). Изначально язык назывался OAK (дуб). Изначально разрабатывался для простейших бытовых приборов. Является строго типизированным языком (тип возвращаемого значения изначально прописан).
Особенность Java: мультиплатформенность («написано однажды, работает везде»). Написанное на Java компилируется в некий код, который затем преобразуется в набор машинных команд на непосредственной машине. Для каждой операционной платформы существует своя JVM (абстрактный компьютер), который превращает этот код в понятный машине байт-код.
