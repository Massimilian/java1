Class Loader (загрузка классов) – осуществляется в несколько этапов:
Загрузка (Loading) – поиск бинарных данных для класса по его имени; сюда входит поиск запрошенного класса среди загруженных ранее, получение байт-кода для загрузки и его проверка на отсутствие ошибок, создание экземпляра класса Class (для работы в runtime), загрузка родительских классов. Если по каким-то причинам родительские классы и интерфейсы не были загружены, класс считается не загруженным.
Связывание, линковка (Linking):
Верификация (Verification) – проверка корректности полученного байт-кода; импортируемого типа или класса (проверка байт-кода на соответствие Java);
Подготовка (Preparation) – выделение памяти для статических переменных класса, выделение памяти под значения по умолчанию – null для объектов, значения по умолчанию для примитивов.
Разрешение (Resolution) – преобразование символьных ссылок на в прямые ссылки; иными словами, разрешение символьных типов, полей и методов.
Инициализация (Initialisation) – вызов Java кода, который инициализирует все переменные с правильными значениями.
Правила загрузки класса:
Класс должен быть сначала полностью загружен, и только потом слинкован.
Класс должен быть полностью проверен и подготовлен, и только потом проинициализирован.
Ошибки разрешения ссылок происходят во время выполнения программы, даже если были обнаружены на этапе линковки.
Существуют три базовых загрузчика класса, каждый из которых осуществляет загрузку из определённого места:
Bootstrap – базовый загрузчик, также называется Primordial ClassLoader. Загружает стандартные классы JDK из архива rt.jar.
Extension ClassLoader – загрузчик расширений. Загружает классы расширений, которые по умолчанию находятся в каталоге jre/lib/ext, но могут быть заданы системным свойством java.ext.dirs.
System ClassLoader – системный загрузчик. Загружает классы приложения, определенные в переменной среды окружения CLASSPATH.
В Java используется иерархия загрузчиков классов, где корневым является базовый, далее следует загрузчик расширений, а за ним системный.
