JDK – Java Development Kid – инструмент разработки Java, который включает в себя компилятор Java (javac), библиотеки классов Java SE, утилиты и документацию. Именно благодаря JDK набранный код превращается в файл .class (тот самый промежуточный код), а затем – в байт-код. Это – главный инструмент разработчика программ на Java. Компилятор (javac) – именно часть JDK.
