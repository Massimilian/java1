Самые распространённые операции со строками:
•	String.toUpperCase() – возвращает строку заглавными буквами.
•	String.toLowerCase() – возвращает строку прописными буквами.
•	String.replace(oldValue, newValue) – замена внутри строки всех значений oldValue на newValue.
•	String.trim() – обрезает все пробелы до и после строки.
•	String.indexOf(value) – возвращает индекс в виде int первого вхождения value. Если подстрока не найдена – вернётся -1.
•	String.subString(integer, end) – возвращает строку, обрезанную с индекса, указанного в аргументе integer, до аргумента end (необязательный аргумент) или конца, если end не указан.
•	String.split(separator) – возвращает массив, который получен из элементов строки, причём разделителем является separator.
•	String.toCharArray() – превращение строки в массив символов.
•	String.contains(value) – булева проверка подстроки value – есть ли она внутри String.
•	String.isEmpty() – булева проверка, что строка пустая. Может работать неустойчиво (например, если строка равна null, то метод вернёт ошибку). Для гарантии работы лучше использовать метод String,equals(“”), который, в случае String == null, вернёт false.
•	String.isBlank() – булева проверка, что строка пустая или содержит только пробелы.
Для соединения нескольких строковых данных лучше использовать StringBuilder и его метод для соединения нескольких строк append. Изначальный размер StringBuilder – 16, но его можно задать в конструкторе любым. Кроме того, StringBuilder можно задать на основе уже существующей строки (также передав её в конструктор). Методы StringBuilder:
•	StringBuilder.append(String) – добавить в конец строку String
•	StringBuilder.insert(offset, String) – вставить в позицию  offset строку String
•	StringBuilder.reverse() – развернуть задом наперёд результат.
