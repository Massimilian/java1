Локальный вложенный внутренний класс - это такой внутренний нестатический класс, который будет создан только внутри работающего метода. Область действия ограничена только тем методом, где создан класс. При этом метод может быть статическим и нестатическим. В этом – глобальное различие вложенного внутреннего класса и локального класса – внутренний класс существует только в условиях своего внешнего класса, а для локального класса существования внешнего класса не обязательно.  Локальный класс видит все свойства и методы внешнего класса (в т.ч. private); все static свойства и методы, если он сам находится в static методе класса. Локальный класс не может быть static, private, protected или public. 
public class Test {
    public Test() { // создаём конструктор
        class Local { // внутри создаём локальный класс
            public void print() {
                System.out.println("We are in the Local");
            }
        }
        new Local().print(); // сразу же запускаем метод локального класса
    }
}

class Main {
    public static void main(String[] args) {
        Test test = new Test(); // при создании класса отработает метод из локального класса Local
    }
}
