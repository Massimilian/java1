Integer (int) занимает он 32 бита, или 4 байта. 
Максимальное значение int 2147483647, минимальное -2147483648. Для оптимизации вычислений данные могут быть неявно преобразованы к int. Вот простой пример: 
byte a = 1;
byte b = 2;
byte result = a + b; // error: incompatible types: possible lossy conversion from int to byte. Придётся поправить.
byte result = (byte)(a + b);
При работе с int значениями остаток отбрасывается, оставляя только целую часть (поэтому в таких случая лучше использовать double).
