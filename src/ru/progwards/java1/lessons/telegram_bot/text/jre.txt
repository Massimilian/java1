JRE – Java Runtime Environment – среда выполнения Java программ (именно благодаря чему и работает принцип «написано однажды, работает везде»). Является минимальной реализацией виртуальной машины. Включает в себя среду выполнения Java программ и библиотеку классов Java SE. Способ работы – скомпилированный байт-код передаётся в Java, где с помощью class Loader-ов загружается в Java Virtual Machine. Перед этим проходится стадия под названием Byte Code Verification (проверка входящего байт-кода на соответствие Java; если всё нормально – Java его выполняет). Также есть некоторые кусочки кода, которые подвергаются JIT (Just In Time) компиляции. Является «плеером» Java-программы. JRE не включает в себя компилятор javac.
